# FuChuangTiKu - 研发岗位（Java后端 + Web前端）知识库

本仓库用于沉淀 Java 后端与 Web 前端岗位面试与知识点内容，格式可直接用于项目内知识库（如静态站点、内部文档系统、检索系统）。

## 目录结构

- `kb.config.yaml`：知识库全局配置
- `index.yaml`：知识库索引与统计
- `data/questions/`：题库（按前端、后端类型拆分）
- `data/examples/`：优秀回答范例
- `data/knowledge_points/`：技术知识点地图

## 数据来源与目标

| 数据类型 | 后端来源 | 前端来源 | 数量要求 |
|---|---|---|---|
| 技术面试题 | 牛客网、CSDN | 牛客网、掘金 | 30题 |
| 项目深挖题 | 面经库 | 面经库 | 10题 |
| 场景设计题 | 技术博客 | 技术博客 | 10题 |
| 行为面试题 | 面经库 | 面经库 | 10题 |
| 优秀回答范例 | 自己整理 | 自己整理 | 每类2-3个 |
| 技术知识点 | 官方文档 | MDN、官方文档 | 不限 |

## 使用建议

1. 按 `data/questions/*.yaml` 持续补充真实题目与答案。
2. 将 `status: draft` 逐步更新为 `status: reviewed`。
3. 在 CI 中校验 `id` 唯一性与字段完整性（可按 YAML schema 扩展）。
4. 检索侧可按 `tags`、`difficulty`、`source` 做过滤。
